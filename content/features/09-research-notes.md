The research and notes module provides a structured repository for reference material gathered during the writing process. Research notes are scoped to a novel and are accessible from the left sidebar below World Building.

Source URLs and links to external references may be embedded directly in the markdown description field using standard markdown link syntax.

## FEAT-0901 Research note list board

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |
| **Extends** | [FEAT-0001 - Entity list board](/novel-it-docs/features/00-base-entity-features#feat-0001-entity-list-board) |

### Description

The research note list board displays all research notes belonging to the current novel. As research notes have no subordinate entities of their own, the board displays research note cards only, with no parent summary section.

## FEAT-0902 Add research note

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |
| **Extends** | [FEAT-0004 - Add entity](/novel-it-docs/features/00-base-entity-features#feat-0004-add-entity) |

### Description

In addition to the properties inherited from the base entity, research notes have the following properties:

- Research note title (overrides base entity Name)
- Type — user-definable category (e.g. Historical, Scientific, Cultural, Legal, Medical)
