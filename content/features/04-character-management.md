## FEAT-0401 Create character

| | |
|---|---|
| **Status** | In Progress |
| **Phase** | 1 |

### Description

The user shall be able to create new characters. Characters shall have the following attributes:
- Full name
- First name
- Middle name
- Last name
- Nickname
- Aliases
- Gender
- Age
- Role in story
- Physical description
- Interview
- The lie they believe
- Arc in story
- Character image
- Relationships
- Appears in scene list

## FEAT-0402 Edit character

| | |
|---|---|
| **Status** | In Progress |
| **Phase** | 1 |

### Description

The user shall be able to click on a character's card on the character dashboard and edit the characters details.

## FEAT-0403 Archive character

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

From the character's details page the user shall be able to archive a character.

Archived characters do not appear on the character dashboard

## FEAT-0404 Unarchive character

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

From the character dashboard the user shall be able to display a list of archived characters and select characters to unarchive.

Once unarchived, the character shall appear on the character dashboard again.

## FEAT-0405 Delete character

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

- From the character's details page the user shall be able to delete a character.
- A warning shall be displayed that this is permanent
- When a character is deleted any manuscript that refers to the character remains unchanged

## FEAT-0406 Character sharing

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 4 |

### Description

- The user can share a character amongst more than one novel
- From the character details page the user can add the additional novels that the character will appear in
- When the user opens the additional novels and navigates to Characters then the character shall be displayed.
- If a shared character is edited then the edits appear in all novels where the character exists
- If a shared character is archived then it is archived for all novels
- If a shared character is deleted then it is deleted for all novels
