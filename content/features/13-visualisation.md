The visualisation module provides graphical views of novel data that complement the board-based navigation. These features are designed to help the author understand relationships and structure at a glance rather than through sequential navigation.

## FEAT-1301 Relationship graph

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 4 |

### Description

The relationship graph provides a visual representation of the relationships between planning entities — characters, locations, items, and world building — and the scenes they appear in.

- The graph shall be accessible from the novel dashboard
- Each entity type shall be represented as a distinct node style to allow visual differentiation
- The user shall be able to toggle each entity type on and off to reduce visual complexity
- The user shall be able to navigate the graph by dragging the canvas
- The user shall be able to zoom in and out of the graph
- Hovering over a node shall display a brief summary popup of that entity
- Clicking on a node shall open that entity's details page
- The user shall be able to search within the graph to locate a specific entity by name
