The plotting seaction of Novel-It allows the user to plan the structure of their novels and then apply this structure to the novel manuscript.

Plots consist of the following elements:
- Acts
- Beats
- Scenes
- Timelines

## FEAT-0801 Plot templates

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

- The user shall be able to choose from predefined templates.
- When a template is chosen then the appropriate plot parts are automatically created in the plotting section of the application.
- The user shall be able to create a custom plot structure.

## FEAT-0802 Plot management

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

- The plot dashboard shall display the acts associated with the plot
- Acts shall be displayed as cards of the dashboard showing the act name and brief details
- Acts can be reordered using drag and drop
- Beats can be moved from one act to another using drag and drop

## FEAT-0803 Act management

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

Acts consist of the following properties:

- Act title
- Description
- When the act details are displayed then the user shall be able to add new beats

### Description

| | |

## FEAT-0806 Act deletion

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

- When on the act details page it shall be possible to delete an act
- When the use'r attempts to delete an act they shall be shown a warning that the deletion is permenant
- When an act is deleted all associated beats and scenes shall be deleted.

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

  - Action overview
  - Associated characters
  - Associated locations
  - Associated items
