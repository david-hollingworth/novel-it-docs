## FEAT-1101 Relationship graphs

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 4 |

### Description

- The user shall be able to display a graph view of the relationships between characters, locations, items and the scenes thy appear in
- The graph view shall allow the user to toggle the different element types on and off
- The user shalll be able to navigate the graph view by dragging the mouse
- The user shall be able to search the graph view for characters, locations and items
- Hovering the mouse over an element in the graph view shall popup a brief details pane
- Clicking on an element in the graph view shall open that element's details.
