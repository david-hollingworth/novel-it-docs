The world building feature forms an addon function to locations and items.  Not every genre of novel will require it. 

## FEAT-0701 Create world-building item

| | |
|---|---|
| **Status** | In Progress |
| **Phase** | 1 |

### Description

The user shall be able to create world-buliding items. They shall have the following attributes

- Name
- Type (seperate from item types) E.g. Religion, culture, Technology Level, Rules and Laws, Historical Timeline
- Description
- Relates to - can relate to character, location, item or other world-building element
- Appears In lists the scenes the word-building item is referenced in.

## FEAT-0702 Edit world-building item

| | |
|---|---|
| **Status** | In Progress |
| **Phase** | 1 |

### Description

The user shall be able to click on a world-building item's card on the world-building item dashboard and edit the world-building item's details.

## FEAT-0703 Archive world-building item

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

From the world-building item's details page the user shall be able to archive a world-building item.

Archived world-building items do not appear on the world-building item dashboard

## FEAT-0704 Unarchive world-building item

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

From the world-building item dashboard the user shall be able to display a list of archived world-building items and select world-building items to unarchive.

Once unarchived, the world-building item shall appear on the world-building item dashboard again.

## FEAT-0705 Delete world-building item

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 2 |

### Description

- From the world-building item's details page the user shall be able to delete a world-building item.
- A warning shall be displayed that this is permanent
- When a world-building item is deleted any manuscript that refers to the world-building item remains unchanged

## FEAT-0706 world-building item sharing

| | |
|---|---|
| **Status** | Draft |
| **Phase** | 4 |

### Description

- The user can share a world-building item amongst more than one novel
- From the world-building item details page the user can add the additional novels that the world-building item will appear in
- When the user opens the additional novels and navigates to world-building items then the world-building item shall be displayed.
- If a shared world-building item is edited then the edits appear in all novels where the world-building item exists
- If a shared world-building item is archived then it is archived for all novels
- If a shared world-building item is deleted then it is deleted for all novels
